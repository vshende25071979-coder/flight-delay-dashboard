# Flight Delay Analysis Dashboard

An interactive dashboard built using Python Dash and Plotly to analyze airline delays.

## Tools Used
- Python
- Dash
- Plotly
- Pandas

## Features
- Carrier Delay Analysis
- Weather Delay Trends
- NAS Delay Analysis
- Security Delay Monitoring
- Late Aircraft Delay Visualization

## Project Output
Interactive year-wise dashboard with dynamic updates using callback functions.

## Run Project

python flight_delay.py

Open in browser:
http://127.0.0.1:8050/
